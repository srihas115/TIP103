## Unit 09: Dynamic Programming III & Binary Trees II

In this unit you'll continue to advance your problem solving skills with binary trees. You'll practice more depth first search traversal problems and learn a new type of traversal: breadth first search.

#### Lecture Session

In lecture, you will explore backtracking and more advanced dynamic programming techniques to handle complex problem spaces.

In this lecture, you will...

- Apply backtracking to explore multiple solution paths
- Combine dynamic programming and recursive strategies for complex problems

#### Breakout Session & HackerRank

In the problem sets, you will...

- Perform breadth first search or level order traversals of a binary tree
- Articulate the differences between a breadth first and depth first search
- Solve intermediate problems that require traversing and manipulating nodes in a tree

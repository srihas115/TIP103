## Unit 05: Graphs II, OOP, & Linked Lists I

Welcome to Unit 5!

#### Lecture Session

In lecture, you will explore graph traversal techniques such as depth-first search and breadth-first search, as well as topological sorting for ordering problems, and how these concepts connect to object-based structures like linked lists.

In this lecture, you will...

-   Traverse graph structures using depth-first and breadth-first search

-   Apply traversal strategies across both graph and linked list problems

-   Apply topological sorting in dependency-based problems

#### Breakout Session & HackerRank

In this unit, you will learn the essentials of object-oriented programming (OOP) with Python through hands-on practice with classes, instances, methods, and attributes. You'll learn how to model real-world entities in your code using classes and to manipulate these entities using methods.

In the problem sets, you will...

-   Define and use classes and objects

-   Use class attributes and methods to manage object data

-   Create and manipulate a singly linked list

-   Solve problems requiring traversal of a singly linked list

### Session 1: Graphs (Traversals, TopoSort) | OOP & Linked Lists

In this session, students will learn to apply Python classes and linked lists through practical exercises. They will begin by creating and manipulating instances of a class and then explore the basics of linked lists, focusing on node creation and linkage. These exercises aim to deepen understanding of object-oriented programming and provide foundational skills in managing custom data structures in Python.

### Session 2: Mock Interview | Linked Lists

In this session, students will delve into the intricate world of linked lists, focusing on both singly and doubly linked list structures. They will engage in a series of hands-on problems that challenge them to perform operations such as insertion, deletion, copying, and value manipulation within linked lists.
